Настройка платежного модуля rficb.
Зарегистрируйтесь в системе rficb (rficb.ru).
В личном кабинете rficb в разделе "Инструменты / Сервисы" добавьте сервис и заполните настройки.
URL скрипта обработчика на Вашем сайте: http://www.example.com/payment/Rficb/callback.php
URL страницы успешной покупки: http://www.example.com/order
URL страницы ошибки: http://www.example.com/order
где вместо www.example.com используйте доменное имя вашего магазина.
Скопируйте папку в корневую папку сайта.

Секретный ключ можно посмотреть в личном кабинете rficb "Инструменты / Сервисы" на странице редактирования сервиса или на странице "Инструменты / API ключи".

В настройках магазина "Настройки->Оплата" создайте способ оплаты, в выпадающем списке выбрать "РФИ Банк", ввести ключ и секретный ключ из личного кабинета.
